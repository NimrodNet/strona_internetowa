Kod Pythona

W internecie możemy napotkać wiele przykładów kodu języka Python.

W bankowości każdy przykład jest dobry.

Poniższy przykład pokazuje rysowanie punktu w języku Python.

import sys
from PIL import Image, ImageDraw

tlo = "rgb(255,255,254)"
rozdzielczosc = (300, 300)

proksy = (100, 100)
punkt = "rgb(0,0,0)"

image = Image.new("RGB", rozdzielczosc, tlo)

rysunek = ImageDraw.Draw(image)
rysunek.point(proksy, punkt)

image.save("wyjscie.png")
image.show()

Program korzysta z popularnego modułu Pillow dla języka Python. Rysuje czarną kropkę na białym tle.

Metoda point() pozwala narysować kropkę. Jest to metoda, którą ktoś już wcześniej stworzył. My możemy z niej korzystać.

Metoda save() zapisuje plik do lokalizacji, a show() pokazuje plik graficzny.

Jest to pokazanie tego, że w prosty sposób można stworzyć grafikę o dowolnej rozdzielczości i zawartości. Można rysować kropki, kreski, figury, ludzi, itd. Fajna rzecz.
